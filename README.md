## Luigi hands-on

This is a hands-on docs for Luigi.

## Requirements
- poetry
- python 3.10^

## Installation
```bash
make setup
```

## How to run

First, run luigid server.
```bash
make run/luigid
```

Next, run task in other terminal.
```bash
make run/task
```
